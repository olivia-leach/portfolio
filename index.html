Olivia Leach, MPH

Full Stack Web & GIS Developer

I was first drawn to code as a way to interact with and analyze data to solve real-world challenges. I am particularly fond of geospatial data, believing that place plays an integral role in determining who we are.

I am excited to leverage my diverse backgrounds in engineering, public health, and web development by positioning myself at the intersection of health and tech—ultimately helping to bridge the innovation gap between these two fields.

When not coding, I love maps, knitting, and being outdoors with my dog!

Web Development
Whereoke

Whereoke

Project Description

Whereoke uses the Google Maps and Google Streetview image APIs to geospatially display Karaoke data based on the day of the week. Users can use CRUD requests to interact with the data, which is seeded on the back end in a PostgresSQL database.

Technologies used:

  • JavaScript
  • HTML 5
  • SCSS
  • Bootstrap
  • Handlebars
  • Ruby on Rails
  • PostgresSQL
  • ActiveRecord

Github: Front End, Back End
